# Omega 跨文本综合 Essays

这一层不再按“单部经典 -> 类别 -> 单章/单卦”展开，而是反过来按 **Omega 定理核** 来追踪不同文本中的结构显现。

目标不是做泛文化评论，而是确认：

1. 同一个形式对象在多部经典中的出现方式
2. 哪些对应是 `formal correspondence`
3. 哪些只应保留为 `metaphorical analogy`
4. 这些结构怎样回到 Gen 2 论文的严格数学语境

当前规划为 `10` 篇旗舰综合文，每篇聚焦一个 Omega 定理核。
