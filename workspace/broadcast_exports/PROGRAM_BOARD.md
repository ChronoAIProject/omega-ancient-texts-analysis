## 当前执行标准

| 模块 | 标准 |
|---|---|
| 选题 | 先看最近同类数据，再选主题；短视频不按易经顺序机械推进 |
| 标题 | 先给现实判断：退、强、空、未来、匹配、过载、折返 |
| Omega 介绍 | 一句话解释，放在 hook 后：Omega 是宇宙笔记看结构如何走向终局的视角 |
| 古典文本 | 易经/道德经作为结构地图，不作为第一秒知识索引 |
| 画面 | 生活图文/公共素材优先；暂停本地卡通；低运动，避免抖动 |
| 公开视频 QA | 发布候选必须抽取开头/中段/后段至少 3 帧检查；四角不能出现内部标签、测试字样、审片说明、工具说明或水印；禁用词包括 `测试`、`审片`、`本地审片版`、`草稿`、`样片`、`内部`、`draft`、`test`、`preview`、`debug`、`QA` |
| 字幕 | 必须和音频同步；发布前截图/播放检查 |
| 声音 | `edge-tts zh-CN-YunyangNeural +0%` |
| 封面 | 必须有核心判断句；不能只是空图或氛围图 |
| Tag | 必须从平台下拉选择结构化话题；不能只在文案里写 `#tag` |
