# Euclid's Elements Baseline

This directory is reserved for curated baseline material for `几何原本 / Euclid's Elements`.

Current use:
- fix the canonical thirteen-book structure
- keep source provenance stable for future article generation
- avoid drifting summaries of the books across different agents

Baseline references used for the current classification:
- Clay Mathematics Institute historical archive: <https://www.claymath.org/library/historical/euclid/index.html>
- Jeremy Avigad, Edward Dean, John Mumma, *A formal system for Euclid's Elements*: <https://arxiv.org/abs/0810.4315>
- Logan Murphy et al., *Autoformalizing Euclidean Geometry*: <https://arxiv.org/abs/2405.17216>
- Stephen Wolfram, *The Empirical Metamathematics of Euclid and Beyond*: <https://arxiv.org/abs/2107.07337>

Planned future additions:
- per-book proposition index
- a compact list of propositions most relevant to Omega backflow
- cleaned public-domain text excerpts for selected books
